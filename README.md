## 📖 這是什麼？

一份結構化整理「違反基礎科學原理」消費產品的開放資料集。

每一筆記錄都包含：產品名稱、宣稱功效、為什麼是智商稅、違反了哪條科學原理、以及市面上常見的價格區間。

> 簡單說：你阿嬤被直銷騙去買什麼，這裡大概都查得到。

---

## 🎯 為什麼要做這個？

1. **資訊不對稱**：廠商用專業術語砸消費者，消費者沒有判斷工具
2. **現有資料太分散**：新聞、PTT、論壇各講各的，沒有一個結構化的開放資料庫
3. **可以當素材**：寫文章、做影片、說服親友、自我進修，都可以用
4. **佔坑**：GitHub 上還沒有人做這個，我們當第一個 ☝️

### 🔥 精選案例

| 產品 | 一句話打臉 |
|------|-----------|
| 膠原蛋白口服液 | 喝下去跟吃一顆蛋沒兩樣，胺基酸不會導航到臉上 |
| 量子手環 | 量子效應只在極低溫發生，你手上的矽膠圈跟薛丁格的貓沒關係 |
| 排毒足貼 | 貼完變黑是竹醋粉遇汗水變色，貼熱水壺上也一樣會黑 |
| 鹼性水 | 喝下去先被胃酸（pH 1.5）中和，跟喝水龍頭的水一樣 |
| 負離子吹風機 | 負離子半衰期極短，吹出來的瞬間就消失了 |
| 信息水 | 水要是有記憶，格式化後的硬碟資料應該還在 |

## 🧰 萬用打臉工具箱

以後看到類似產品，套這四題就好：

### 1. 看到「排毒」→ 問三題
- 什麼毒？化學式？
- 排出多少？用什麼儀器量的？
- 知道人體有肝臟跟腎臟嗎？

### 2. 看到「量子／能量／共振／頻率」→ 問三題
- 哪個物理量？
- 單位是什麼？
- 用什麼儀器量的？

### 3. 看到「小分子／奈米／離子／活性」→ 問三題
- 分子量多少道爾頓？
- 粒徑分佈呢？
- 半衰期多久？

### 4. 看到「XX提取物／XX胜肽」→ 問三題
- 有效成分濃度？
- 透皮吸收率？（外用）
- 有人體臨床實驗嗎？

## 🤝 貢獻方式

歡迎任何人補充！你需要做的：

1. **新增產品** → 在 `data/products.csv` 加一行，發 PR
2. **補充資料來源** → 填上 `參考資料連結` 欄位
3. **討論／勘誤** → 開 Issue 討論
4. **翻譯** → 歡迎 i18n

詳見 **[CONTRIBUTING.md](CONTRIBUTING.md)**

### 貢獻格式速查

```csv
ID,分類,產品名稱,宣稱功效,打臉原因,違反原理,常見價格區間(台幣),關鍵字Tags,參考資料連結,備註,加入日期
25,物理學,你的新產品,廠商宣稱,科學打臉,違反什麼,價格,標籤1;標籤2,網址,備註,2026-06-09
```
